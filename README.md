# AI 실무 협업 | 개인 브랜딩 랜딩 페이지

AI 활용이 막막한 직장인을 위한 **실무 AI 협업** 개인 브랜딩용 모바일 랜딩 페이지입니다.  
HTML, CSS, JavaScript만으로 구현했으며, 밝은 톤의 모바일 퍼스트 UI로 제작되었습니다.

## 개요

실무에 바로 적용 가능한 AI 협업 방법을 소개하고, 업무 생산성 극대화와 창의적인 일에 집중하는 즐거움을 전달하는 랜딩 페이지입니다.

## 기술 스택

- **HTML5** — 시맨틱 마크업
- **CSS3** — 모바일 퍼스트, 반응형 레이아웃
- **JavaScript (Vanilla)** — 스크롤 애니메이션, CTA 버튼 처리

## 프로젝트 구조

```
pr/
├── index.html   # 메인 페이지 (히어로, 주제 카드, 커피챗 CTA, 푸터)
├── styles.css   # 스타일 (밝은 톤 팔레트, 카드/버튼)
├── script.js    # 인터랙션 (스크롤 페이드인, 커피챗 링크)
└── README.md    # 프로젝트 설명
```

## 콘텐츠 구성

1. **히어로** — 헤드라인 + 서비스 개요 문구  
2. **주제 카드 3개**
   - AI 워크 스마트: 실전 AI 활용법 & 성공 사례  
   - AI 동료 만들기: No-code/Low-code 업무 자동화  
   - AI로 데이터 인사이트 발견: 데이터 분석 & 시각화  
3. **커피챗 CTA** — 구글 폼 설문지 링크 버튼  
4. **푸터** — 저작권 문구  

## 실행 방법

### 방법 1: 로컬 서버 (권장)

```bash
cd pr
npx serve -l 3000
```

브라우저에서 **http://localhost:3000** 접속.

### 방법 2: 파일 직접 열기

`index.html` 파일을 더블클릭하여 브라우저에서 열기.

## 커스터마이징

- **커피챗 링크 변경**: `index.html`에서 `#coffee-chat-btn`의 `href` 값을 원하는 구글 폼(또는 외부) URL로 수정하면 됩니다.  
  현재 연결: [커피챗 사전 설문지](https://forms.gle/WTfFbSbi6kZCXqSs8)
